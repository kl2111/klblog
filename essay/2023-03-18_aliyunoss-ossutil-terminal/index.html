1. 安装Xcode Command Line Tools#

Xcode是一款由苹果公司开发的集成开发环境（IDE），主要用于开发MacOS、iOS、iPadOS、watchOS和tvOS应用程序。在MacOS中，您需要先安装Xcode Command Line Tools才能使用ossutil上传工具。您可以通过以下步骤安装：

  1. 打开终端工具（Terminal）。
  2. 输入以下命令，然后按回车键：
xcode-select --install
  1. 稍等片刻，系统将自动下载并安装Xcode Command Line Tools。

2. 安装ossutil命令行工具#

您可以通过以下步骤安装ossutil命令行工具：

  1. 打开终端工具（Terminal）。
  2. 输入以下命令，然后按回车键：
sudo -v ; curl https://gosspublic.alicdn.com/ossutil/install.sh | sudo bash
  1. 稍等片刻，系统将自动下载并安装ossutil命令行工具。
  2. 如果您想检查ossutil是否安装成功，请在终端中输入以下命令：
ossutil --version

如果显示版本信息，则说明ossutil安装成功。

3. 配置ossutil命令行工具#

在使用ossutil上传文件之前，您需要先配置ossutil命令行工具，包括AccessKey ID、AccessKey Secret和Bucket的endpoint。建议使用子AccessKey，并在Bucket中为子AccessKey赋予对应的权限。

您可以通过以下命令进行配置：

ossutil config

然后，按照提示输入您的AccessKey ID和AccessKey Secret即可完成配置。sts token默认为空，无需输入。在配置endpoint时，您可以在阿里云OSS控制台中找到您的Bucket，然后在“基础设置”页面查看Bucket的endpoint地址。

4. 上传文件到OSS#

1)您可以使用以下命令将文件上传到指定的Bucket：

ossutil cp <local-file-path> oss://<bucket-name>/<object-key-prefix>

其中，<local-file-path>表示本地文件的路径，<bucket-name>表示OSS存储空间的名称，<object-key-prefix>表示在OSS存储空间中存储文件的路径和文件名，如果是放到根目录就不需要。

2）如果您需要上传本地文件夹中的所有文件和子文件夹，并开启覆盖权限，可以使用以下命令：

ossutil cp -rf <local-folder-path> oss://<bucket-name>/<object-key-prefix>

例如：ossutil cp -rf /Users/ohyoung/project/blog/kailiang-life/public oss://kl-blog

其中，-r参数表示上传文件夹中的所有文件和子文件夹，-f参数表示强制覆盖Bucket中已有的对象。若上传上传文件夹中的所有文件和子文件夹但是否覆盖需要确认，删除-f即可。
